Duża fabryka samochodów elektrycznych wprowadza ulepszenia do swojego systemu autopilota i potrzebuje Państwa pomocy we wdrożeniu programu, który zdecyduje, czy samochód B, który jedzie pośrodku dwóch samochodów A i C, musi przyspieszyć, zwolnić lub utrzymać bieżącą prędkość. Samochody są takie same, a czujniki autopilota podadzą aktualną pozycję tylnej części trzech samochodów na dowolnej osi. Proszę zobaczyć przykład na rysunku.

Samochód B należy przyspieszyć, jeśli odległość od jego tyłu do tyłu samochodu A jest mniejsza niż odległość od jego tyłu do tyłu samochodu C. Jeśli jest większa, należy go zwolnić. Jeśli jest taka sama, należy utrzymać aktualną prędkość.

Dane wejściowe: Trzy liczby całkowite ({% if is_js %}number{% else %}int{% endif %})które reprezentują aktualne pozycje tyłów samochodów A, B i C, odpowiednio.

Wyjście: Liczba całkowita 1, jeśli samochód B musi przyspieszyć; -1, jeśli musi zwolnić; lub 0, jeśli musi utrzymać bieżącą prędkość.
